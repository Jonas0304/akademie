## 🎓 Ausbildungs-Anfragen verwalten

**Für Leitungspersonen:**

### Panel posten:
```
/anfrage_panel
```
Postet das Anfrage-Dropdown in den aktuellen Kanal.

### Abteilung deaktivieren:
```
/anfrage_toggle bereich: [Name] aktion: deaktivieren
```

### Abteilung aktivieren:
```
/anfrage_toggle bereich: [Name] aktion: aktivieren
```

Verfügbare Abteilungen:
• Theoretische Grundausbildung
• Grundausbildung
• StVO Ausbildung
• Gehobener Dienst
• Fortbildung Waffen
• Fortbildung Verhöre
• Fortbildung Motorrad
• Fortbildung Recht Rhetorik
• KRIPO Ausbildung
• SEK Ausbildung
• VD Fortbildung
• Taktische Notfallmedizin

📌 Deaktivierte Abteilungen erscheinen nicht im Dropdown-Menü!
